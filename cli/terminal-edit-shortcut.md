## 라인, 단어 단위 편집

라인 전체 지우기: ctrl + u

커서 이전 단어 지우기: ctrl + w

커서 이후 전체 지우기: ctrl + k

커서 이전 문자 지우기(backspace 역할): ctrl + h

커서 이후 문자 지우기(delete 역할): ctrl + d


## 커서 이동

라인 처음으로 이동: ctrl + a

라인 끝으로 이동: ctrl + e

커서 이전 단어로 이동: alt + b 또는 esc + b

커서 다음 단어로 이동: alt + f 또는 esc + f


## 화면, 스크롤 이동

화면 클리어: cmd + k

모든 라인 지우기: ctrl + l

스크롤 올리기: cmd + ↑

스크롤 내리기: cmd + ↓


## 명령 히스토리

이전 명령: ↑

다음 명령: ↓

단어 기반 히스토리 검색: ctrl + r

검색 종료: ctrl + g


## 프로세스 제어

실행 중단(SIGINT): ctrl + c

일시정지(SIGSTP): ctrl + z

재개(포그라운드): fg

백그라운드 실행: bg


## 자동완성, 기타

탭 자동완성: tab 또는 ctrl + e

탭 자동완성 옵션 순환: tab 연속 입력


## iTerm2 단축키

출력 강제 새로고침: cmd + r

팬 가로 분할: cmd + d

팬 세로 분할: cmd + shift + d

팬 이동: cmd + option + 방향키

탭 이동: cmd + 숫자
